# NCLEX Practice App

A personal study tool built to help prepare for NCLEX nursing exams.

## ⚠️ Important Notice

This application is a **private study tool** intended for personal use only.

The practice questions in this app are derived from:
> Sole, M.L., Klein, D.G., & Moseley, M.J. (2021). *Introduction to Critical Care Nursing* (8th ed.). Elsevier.

**This site is not publicly distributed or indexed.** If you have accessed it without the owner's permission, please do not use or share it.

## About

A browser-based NCLEX practice application with multiple question types, per-question rationales, topic filtering, and progress tracking.
